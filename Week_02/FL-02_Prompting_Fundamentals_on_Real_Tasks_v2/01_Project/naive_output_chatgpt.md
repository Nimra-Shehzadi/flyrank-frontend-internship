# Responsive Settings Form

A simple, responsive **Settings Form** built using **HTML, CSS, and JavaScript**. The project demonstrates basic form creation, client-side validation, and responsive web design.

## Features

- Responsive layout for desktop and mobile devices
- Full Name input (required)
- Email Address input (required with email validation)
- Theme selection (Light/Dark)
- Enable Notifications checkbox
- Save Settings button
- Client-side form validation
- Clear error messages for invalid inputs
- Success message after successful form submission

## Project Structure

```
settings-form/
│── index.html
│── style.css
│── script.js
└── README.md
```

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## Validation Rules

- Full Name cannot be empty.
- Email Address cannot be empty.
- Email Address must follow a valid email format.
- Form is submitted only when all required fields are valid.

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in any modern web browser.
4. Fill in the required fields.
5. Click **Save Settings** to validate and submit the form.

## Expected Output

- Displays validation errors for invalid or empty required fields.
- Displays a success message when the form passes all validations.
- Provides a responsive user interface across different screen sizes.
