This is a working, responsive Settings Form with:

Full Name and Email fields, both required
Theme dropdown (Light / Dark / System)
Notifications checkbox
Save Changes button
Validation that runs on submit, with inline error messages under each invalid field
A success banner that appears once everything checks out, and clears itself if you resubmit

Errors clear live as you fix each field, so people aren't stuck staring at a red border after they've already corrected it.
