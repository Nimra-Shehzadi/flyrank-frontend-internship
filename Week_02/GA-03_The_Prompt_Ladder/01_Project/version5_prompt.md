# Version 5 Prompt (Verification & Self-Check)

Create a responsive Settings Form using HTML, CSS, and JavaScript.

Requirements:
- Full Name (required)
- Email Address (required)
- Theme Selection (Light/Dark)
- Enable Notifications checkbox
- Save Settings button
- Validate required fields
- Show success message

Audience:
This project is for beginner frontend developers who are learning HTML, CSS, and JavaScript.

Constraints:
- Use only HTML, CSS, and JavaScript.
- Do not use Bootstrap or any frontend framework.
- Keep the code clean and beginner-friendly.

Output Format:
Return the solution in three separate files:
- index.html
- style.css
- script.js

Quality Criteria:
- Use semantic HTML.
- Make the layout fully responsive.
- Add comments where appropriate.
- Keep the code readable and well organized.
- Use accessible labels and inputs.

Verification:
Before finishing, verify that every requirement has been implemented. At the end, provide a checklist showing which requirements have been completed and briefly explain how each one was implemented.
