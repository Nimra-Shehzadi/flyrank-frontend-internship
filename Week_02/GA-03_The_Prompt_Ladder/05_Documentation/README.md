# GA-03 – The Prompt Ladder

## Objective
Learn how iterative prompt improvements lead to higher-quality AI responses.

## Work Completed
- Created a baseline prompt.
- Improved the prompt through five iterations.
- Documented each improvement.
- Compared the quality of each AI response.
- Summarized the overall learning.

## Outcome
This exercise demonstrated that structured prompts produce more accurate, organized, and maintainable AI-generated solutions.
