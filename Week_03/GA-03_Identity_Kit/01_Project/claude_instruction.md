# Claude Project Instruction

Always use the following visual identity when generating designs or frontend code:

- Heading Font: Poppins
- Body Font: Inter
- Primary Color: #3B82F6
- Background: #F8FAFC
- Text Color: #111827
- Accent Color: #06B6D4
- Style: Clean, modern, minimal, professional, and easy to read.
- Use generous spacing, rounded corners, and consistent typography across all pages.
