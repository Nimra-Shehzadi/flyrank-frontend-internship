# The Through-Line: Map Content & CTAs

## Objective

This assignment focuses on planning the structure of a professional portfolio website by defining a clear message, organizing page content, and creating effective calls to action.

## Deliverables

- One-line portfolio claim
- Website content map
- Call-to-action (CTA) plan
- List of remaining assets to collect

## Outcome

A structured content plan that will guide the development of the portfolio website and ensure a consistent user experience.
