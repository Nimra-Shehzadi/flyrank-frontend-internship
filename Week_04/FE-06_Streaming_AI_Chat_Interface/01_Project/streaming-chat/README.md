# Streaming AI Chat Interface

## Overview

This project is a modern AI chat interface built using React, TypeScript, and Vite. It simulates an AI conversation with a streaming response effect similar to ChatGPT.

## Features

- Modern chat interface
- User and assistant message bubbles
- Simulated streaming AI responses
- Typing indicator
- Auto-scroll to latest message
- Stop streaming button
- Responsive layout

## Technologies Used

- React
- TypeScript
- Vite
- CSS

## Project Structure

```text
src/
├── App.tsx
├── main.tsx
├── index.css
```

## How to Run

```bash
npm install
npm run dev
```

Open:

```
http://localhost:5173
```
