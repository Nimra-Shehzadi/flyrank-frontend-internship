# Empty but Live

## Overview

This project is a minimal React + TypeScript application created using Vite as part of the FlyRank Frontend AI Engineering Internship.

The purpose of this assignment is to deploy a simple live website and verify that it is publicly accessible before beginning the full capstone development.

---

## Features

- Minimal React application
- Built with React + TypeScript + Vite
- Live deployment on Vercel
- Responsive full-screen layout
- Simple project identity page

---

## Technologies Used

- React
- TypeScript
- Vite
- CSS

## How to Run

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open:

```
http://localhost:5173
```

---

## Deployment

This project is deployed using **Vercel**.

The live deployment confirms that the application is publicly accessible and ready for future development.
