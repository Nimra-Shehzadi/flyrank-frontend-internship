# Ship an Automation Workflow v2

## Overview

This project demonstrates an AI-assisted workflow that automates common documentation tasks for software projects.

The workflow generates:

- Project descriptions
- Feature lists
- README documentation
- LinkedIn post drafts

The workflow reduces repetitive manual work while keeping a human review step before publication.

---

## Workflow

1. Receive project idea
2. Generate project description
3. Generate features
4. Generate README
5. Generate LinkedIn post
6. Human review
7. Publish

---

## Deliverables

- workflow.md
- prompts.md
- workflow-diagram.png
- runs.md
- review.md

---

## Estimated Time Saved

Approximately 60 minutes per project.
