# AI Prompts Used in the Workflow

## Prompt 1 – Project Description

Create a professional project description for the following project.

Include:
- Purpose
- Problem it solves
- Technologies used
- Benefits

Return in professional GitHub README format.

---

## Prompt 2 – Features

Generate a professional features list from these notes.

Return only bullet points.

---

## Prompt 3 – README Generator

Create a complete README including:

- Project Overview
- Features
- Technologies
- Installation
- Folder Structure
- Future Improvements
- License

Use professional markdown formatting.

---

## Prompt 4 – LinkedIn Post

Write a professional LinkedIn post announcing my project.

Include:
- Hook
- Short project summary
- Technologies
- Call-to-action
- Relevant hashtags

Tone:
Professional, clean, and engaging.

---

## Prompt 5 – Human Review Checklist

Before publishing, verify:

- Grammar
- Technical correctness
- Markdown formatting
- Missing information
- Professional tone
