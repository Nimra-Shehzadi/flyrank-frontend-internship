# Three Roads: Choosing My Tech Stack

## Overview

This assignment compares three frontend development stacks and explains the reasoning behind selecting the most suitable option for my portfolio website.

---

## Project Constraints

- Budget: Free only
- Skill Level: Intermediate Frontend Developer
- Portfolio Goal:
  - Projects
  - Skills
  - Resume
  - Contact
  - Live Demo
- Backend Required: No

---

## Technology Options

### Option 1 — HTML, CSS & JavaScript

Pros

- Beginner friendly
- Lightweight
- Easy to start

Cons

- Difficult to maintain
- No reusable components

---

### Option 2 — React + Vite ✅ (Selected)

Pros

- Component-based architecture
- Fast development
- Easy maintenance
- Modern ecosystem
- Professional portfolio structure

Cons

- Small learning curve

---

### Option 3 — Next.js

Pros

- Excellent SEO
- Highly scalable
- Powerful framework

Cons

- More complex
- Unnecessary for the current project

---

## Final Decision

I selected **React + Vite** because it is free, modern, scalable, easy to maintain, and matches my current frontend development skills. It is the best choice for building a professional portfolio website.
